题目与题解

T1：模型

Time Limit: 1000 ms Memory Limit: 256MB

题意：

一天黎瑟去看望诺莉塔，从自己的作业里找了一个树的模型带给她。

一起玩的时候诺莉塔一不小心摔碎了树的模型。

黎瑟回去修复了模型，为了不让模型太脆弱，她决定在树上加一些边，使得图中没有割点。

求一组所需加边最少的方案。

题解：

我们可以先考虑如果是割边的话应该怎么做。显然答案下界是 ⌈l/2⌉ 其中 l 是树中的叶节点个数。我们可以构造出这样的方案：令叶节点的权值为 1 ，非叶节点的权值为 0 ，求整棵树的带权重心 cc 。由于重心的每个儿子的子树里都只有不超过 ⌊c/2⌋ 个叶子，我们不难构造出一组方案使得每条边连接的两个叶子都来自于重心的不同儿子的子树。

考虑割点，显然答案下界是 max(max{di−1},⌈l/2⌉) ，其中 di 是点 i 的度数。我们可以构造出这样的方案：注意到之前的方案里除了重心之外，每个点都保证了删掉它后，它的所有儿子的子树依然是与重心所在的连通块连通的，固整个图也是连通的。

题意：

有1−n一共n个数。保证n为偶数。

小M要把这2m个数两两配对,一共配成m对。每一对的权值是他们两个数的和。

小M想要知道这m对里最大的权值的期望是多少。可怜的小M当然不知道啦，所以她向你求助。

请输出答案对10^9+7取模的值。

T3：Mythological VII

题意：

数轴上有一行n个点，每个点有权值。小M和小S在上面玩游戏。

小M和小S一开始都在第kk号点上，小M只能向左移动，小S只能向右移动。

令px表示当前时刻小M的位置,py表示当前时刻小S的位置，要求任意时刻都满足(px,py]这个区间的所有数和都≤0。

现在，小M想要知道px,py能否同时等于1,n。可怜的小M当然不会啦，所以她向你求助。

题解：

（我并没有看懂100分的做法，所以我写了80分的大错特错大骗分的贪心）

这种贪心就是：对于当前所在的左右端点x, y，我们在x左边找到最远能走到哪，然后在这个范围内找sum的最小值（sum为做过的路），然后右边也一样，取这两边的最小值的最小值，然后移到那个位置。如果走到了1和n，就是yes，反之就是no。
